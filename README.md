# Nutrivae DSA

JavaScript notes and practice solutions for data structures, algorithms, and common coding interview problems.

## Contents

- Algorithm notes and time complexity examples in the project root.
- Problem solutions grouped by topic under `problems/`.
- Simple runnable JavaScript files with example `console.log` checks.

```bash
node problems/Array/1_two_sum.js
node problems/Array/15_three_sum.js
```
